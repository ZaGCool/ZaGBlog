扩展

range(start, stop) 函数可创建一个整数int列表 一般用在for循环中

  • start 开始 默认为0 比如 range(5) 等价 range(0,5)
  • stop 结束 但不包括stop 例如 range(0,5) 是[0,1,2,3,4]

break和continue的区别

break 推出整个循环 continue推出当前循环 它们只能退出一层循环
